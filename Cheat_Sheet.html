Formatting

Generally CSS should be used but using style attribute a lot of basic formatting could be achieved.Also using tags such as strong and em also helps. Here is a list of all such formatting elements.

  • strong - Important[Bold] text
  • em - Emphasized text
  • mark - Marked text
  • small - Smaller text
  • del - Deleted text
  • ins - Inserted text
  • sub - Subscript text
  • sup - Superscript text
